## 🚀 Features

- 🔑 **Save your OpenWeather API key** (stored locally)
- 🔍 **Search any city** – get current weather + 5‑day forecast
- 🌡️ **Temperature (°F)**, wind speed, humidity
- ☀️ **UV Index** with color‑coded safety levels
- 📜 **Search history** – click to revisit cities
- 📱 **Fully responsive** – works on desktop & mobile

---

## 🛠️ Built With

- **HTML5** – structure  
- **CSS3** – gradients, glassmorphism, responsive grid  
- **JavaScript (ES6)** – async/await, localStorage, dynamic DOM  
- **OpenWeather API** – Geocoding + One Call 3.0
